Visual Language (Beeldtaal)

Description

Visual Language was a course mostly about the theory surrounding Visual Interface Design. We learned about Semantics, Rethorics, Gestalt and how to use them in our designs.

Product

During this course I created several visuals which involved either Semantics, Rethorics or Gestalt.

Below you can view the design files including research and feedback sessions.

Semantics

For Semantics we created three visuals in which we set three different moods and meanings using semantics.

Rethorics

To learn more about Rethorics I picked an awareness poster from Amnesty International. I researched the poster and made a potential "upgrade" to it.
