Typography (Typografie)

Description

The course Typography was all about learning the basics of typography. We learned about the different types of fonts, how to use them and how to combine them. We also learned about the effect typography has on the reader and how to use it to your advantage.

Assignments

This course consisted of two assignments. We first created a poster of a specific font we found interesting. The second assignment was to create a booklet in which we demo a font.

At the start of this typography course we took a stroll through Amsterdam to look at different types of fonts and where/how they're being used. We took a collection of pictures which we would later use to pick a font from.

During this stroll I came across a poster from the Moco Museum in Amsterdam. Their logo uses a front that I found to be quite interesting. This was the font I decided to pick for my assignments.

Below is the poster I created for the first assignment. I decided to go with the word "lush" (We had to choose a four letter word for the poster) because I felt like it would fit the font.
