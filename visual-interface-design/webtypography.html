Web typography (webtypografie)

Description

Web Typography was all about creating content that's accessible. This assignment put focus on creating content for deaf people. Right now the standard for deaf people is to have closed captions on the bottom of the screen. This is however quite boring for the viewer and doesnt convey emotion as well as it would for a hearing person.

Finished product

We got assigned a clip from the movie Blade Runner 2049 to which we added more visuals and animations to make watching a movie more interesting for a deaf person.
