Generic dressup game

html - css - javascript

Description

In this project I worked on creating a dress up game with Javascript. The game is contained in what looks like an oldschool game website including the annoying ads and popups. The goal of this project was to create something with Javascript that would provide interaction.

Process

For this game I went through several versions before I got to the final design. At first I wanted to create a Covid-19 themed dressup game but I ended up not liking it all that much and switched to a nostalgic game design.

In this game you can dress up your character in multiple styles and move away the menu for a screenshot when you're finished.
