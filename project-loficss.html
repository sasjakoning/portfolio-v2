Lo-fi CSS girl

html - css

Description

During this project I put together a 2d movable image fully in css. I did this by putting together many html elements and styling these to fit my design. This project greatly helped me improve my css skills.

Process

While the background of the above image is hand-drawn, the girl is completely made out of html/css objects. By giving these objects the correct hierarchy, I could bring this girl to life by animating her. She types on her keyboard, writes notes and blinks.

I started off with a sketch and started laying down the building blocks in html/css.

I drew two different backgrounds that the user could switch between and I added lo-fi music to add mood.
