International Student Guide Amsterdam - A webapp

html - css

Description

During this project I created a webapp in addition to the International Student Guide Amsterdam. This app gives the user multiple cards with statements that could be either fact or fiction. By pressing the cards they turn around and reveal the answers. This makes it a fun and engaging way for the user to learn more about the Netherlands and Amsterdam.

Process

This project required me to work within a certain style. The fonts and colors were set. I designed my own logo and drew all the images for the cards.

My biggest challenge for this project was to figure out ways to make the app interactive without using Javascript. Turning a card around when the user presses it is something easily done with JS but having to do it with Css was a great challenge. It did help me learn more about Css and the tricks and hacks you can do with it.
